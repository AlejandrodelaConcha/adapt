Regular Transfer for Binary Linear Classification

RegularTransferLC is a parameter-based domain adaptation method.

The method is based on the assumption that a good target estimator can be obtained by adapting the parameters of a pre-trained source estimator using a few labeled target data.

The approach consist in fitting a linear estimator on target data according to an objective function regularized by the euclidean distance between source and target parameters:

\[\beta_T = \underset{\beta \in \mathbb{R}^p}{\text{argmin}} \, \ell(\beta, X_T, y_T) + \lambda ||\beta - \beta_S||^2\]

Where:

  • \(\ell\) is the log-likelihood function.

  • \(\beta_T\) are the target model parameters.

  • \(\beta_S = \underset{\beta \in \mathbb{R}^p}{\text{argmin}} \, \ell(\beta, X_S, y_S)\) are the source model parameters.

  • \((X_S, y_S), (X_T, y_T)\) are respectively the source and the target labeled data.

  • \(p\) is the number of features in \(X_T\) (\(+1\) if intercept is True).

  • \(\lambda\) is a trade-off parameter.

Examples

>>> import numpy as np
>>> from adapt.parameter_based import RegularTransferLC
>>> from sklearn.linear_model import LogisticRegression
>>> np.random.seed(0)
>>> Xs = np.random.randn(50) * 0.1
>>> Xs = np.concatenate((Xs, Xs + 1.))
>>> Xt = np.random.randn(100) * 0.1
>>> ys = (Xs < 0.1).astype(int)
>>> yt = (Xt < 0.05).astype(int)
>>> lc = LogisticRegression()
>>> lc.fit(Xs.reshape(-1, 1), ys)
>>> lc.score(Xt.reshape(-1, 1), yt)
0.67
>>> rt = RegularTransferLC(lc, lambda_=0.01, random_state=0)
>>> rt.fit(Xt[:10], yt[:10])
>>> rt.estimator_.score(Xt.reshape(-1, 1), yt)
0.67
