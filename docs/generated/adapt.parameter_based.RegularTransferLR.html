adapt.parameter_based.RegularTransferLR¶

class adapt.parameter_based.RegularTransferLR(estimator, lambda_=1.0, copy=True, random_state=None)[source]¶

Regular Transfer with Linear Regression

RegularTransferLR is a parameter-based domain adaptation method.

The method is based on the assumption that a good target estimator can be obtained by adapting the parameters of a pre-trained source estimator using a few labeled target data.

The approach consist in fitting a linear estimator on target data according to an objective function regularized by the euclidean distance between source and target parameters:

\[\beta_T = \underset{\beta \in \mathbb{R}^p}{\text{argmin}} \, ||X_T\beta - y_T||^2 + \lambda ||\beta - \beta_S||^2\]

Where:

  • \(\beta_T\) are the target model parameters.

  • \(\beta_S = \underset{\beta \in \mathbb{R}^p}{\text{argmin}} \, ||X_S\beta - y_S||^2\) are the source model parameters.

  • \((X_S, y_S), (X_T, y_T)\) are respectively the source and the target labeled data.

  • \(p\) is the number of features in \(X_T\) (\(+1\) if intercept is True).

  • \(\lambda\) is a trade-off parameter.

Examples

>>> import numpy as np
>>> from adapt.parameter_based import RegularTransferLR
>>> from sklearn.linear_model import LinearRegression
>>> np.random.seed(0)
>>> Xs = np.random.randn(50) * 0.1
>>> Xs = np.concatenate((Xs, Xs + 1.))
>>> Xt = np.random.randn(100) * 0.1
>>> ys = (np.array([-0.2 * x if x<0.5 else 1. for x in Xs])
...       + 0.1 * np.random.randn(100))
>>> yt = 0.75 * Xt + 0.1 * np.random.randn(100)
>>> lr = LinearRegression()
>>> lr.fit(Xs.reshape(-1, 1), ys)
>>> lr.score(Xt.reshape(-1, 1), yt)
0.2912...
>>> rt = RegularTransferLR(lr, lambda_=0.01, random_state=0)
>>> rt.fit(Xt[:10], yt[:10])
>>> rt.estimator_.score(Xt.reshape(-1, 1), yt)
0.3276...
